⏰ Relógio Digital

Uma aplicação de Relógio Digital desenvolvida com HTML, CSS e JavaScript, criada para praticar manipulação do DOM, 
atualização em tempo real e funções de data e hora da linguagem JavaScript.

🚀 Funcionalidades
🕒 Exibição da hora em tempo real
📅 Atualização automática das informações
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (ES6)
🎯 Objetivo do projeto

Este projeto foi desenvolvido para reforçar conhecimentos em JavaScript, manipulação do DOM e atualização automática de elementos da página.

Durante o desenvolvimento foram praticados conceitos como:

Manipulação do DOM
Funções
Eventos
